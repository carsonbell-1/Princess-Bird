# HTML Website

Skeleton folder structure for a static HTML website.

Folders:
- src/css - styles
- src/js - scripts
- src/images - images and assets
- src/fonts - font files
- src/components - reusable HTML partials
- pages - top-level HTML pages
- dist - build output
